你现在需要清洗宏观数据，并生成标准化的月频宏观因子面板。

项目结构如下（不要假设其它路径存在）：

project/
    raw_data/
        宏观数据.xlsx
    processed_data/
        （已存在 asset_price_panel.csv、asset_return_panel.csv）

你的任务：

生成文件：
processed_data/macro_panel.csv

宏观因子必须包含（列名统一为英文）：
- PMI
- CPI（当月同比）
- CN10Y（中国10年期国债到期收益率：10年）

注意：宏观数据是“倒序排列”，且含有多行表头/指标信息等“脏行”，必须避免把脏东西洗进去。

要求如下：

------------------------------------------
1）读取与定位数据区域
------------------------------------------
读取 raw_data/宏观数据.xlsx

该文件中可能存在：
- 指标名称、频率、单位、来源、指标ID 等说明行（不属于数据）
- 真正的数据从“时间”这一列开始，时间格式可能是 YYYY-MM 或类似（例如 2026-02）

你必须：
- 只保留真正的数据行（时间能被解析为年月的行）
- 丢弃所有无法解析为日期的行

------------------------------------------
2）时间字段标准化
------------------------------------------
将“时间”字段统一命名为：date

date 必须是 pandas datetime（月频），并统一落在“月末”：
例如 2016-01 -> 2016-01-31

（提示：可用 pd.to_datetime(...) + MonthEnd(0)）

然后按 date 升序排序（从早到晚）。

------------------------------------------
3）列选择与重命名
------------------------------------------
从表格中定位并提取以下三列（允许原列名包含中文，只要能唯一匹配）：

- PMI：制造业PMI
- CN10Y：10年期国债收益率（中债/到期收益率:10年）
- CPI：CPI 当月同比

提取后统一重命名为：

date
PMI
CPI
CN10Y

------------------------------------------
4）数值清洗（必须严格）
------------------------------------------
对 PMI / CPI / CN10Y 三列：

- 去除百分号、逗号等符号
- 强制转为 float
- 无法转为 float 的设为 NaN
- 不允许 forward fill / backward fill
- 允许少量 NaN 保留

------------------------------------------
5）时间截取
------------------------------------------
只保留 date >= 2016-01-01 且 date <= 2026-02-28 的数据

（因为你的资产数据覆盖到 2026-02-27）

------------------------------------------
6）质量检查（必须打印）
------------------------------------------
打印：

A. 宏观数据起始/结束月份
B. 各列缺失值统计
C. 是否存在重复月份（如果存在，打印重复月份并去重：保留最后一条）
D. 关键月份检查：
   - 打印 2016-01、2018-01、2024-01、2025-12（如果存在）的三因子值

------------------------------------------
7）保存
------------------------------------------
保存为：
processed_data/macro_panel.csv

保存要求：
- date 为第一列（月末日期）
- 不保存 index
- 升序排列
- 保留 NaN

------------------------------------------
代码风格要求
------------------------------------------
写成一个脚本文件：build_macro_panel.py
结构函数化：load -> clean -> validate -> save
只使用 pandas / numpy（如需）
