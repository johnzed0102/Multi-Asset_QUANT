你现在需要基于已生成的价格面板，构建“日收益率面板”。

项目结构如下（不要假设其它路径存在）：

project/
    raw_data/
    prompt/
    processed_data/
        asset_price_panel.csv

你的任务：

生成文件：
processed_data/asset_return_panel.csv

要求如下：

------------------------------------------
1）读取价格面板
------------------------------------------
读取 processed_data/asset_price_panel.csv
确保 date 列解析为 datetime
按 date 升序排序
去除重复日期（如存在）

价格列应为以下 6 个（若顺序不同请按列名选择）：
hs300
sp500
cgb10y
gold
policy_bond
credit_bond

------------------------------------------
2）收益率计算规则（必须严格执行）
------------------------------------------
对每个资产列分别计算“对数收益率”：

ret_t = ln(P_t / P_{t-1})

注意：
- 不能用简单收益率
- 不能 forward fill / backward fill
- 保留 NaN（特别是 credit_bond 在 2022-07-01 之前应保持 NaN）
- 如果价格为 0 或负数（理论上不应出现），该行收益率设为 NaN，并打印告警统计

------------------------------------------
3）输出结构
------------------------------------------
输出 DataFrame 列顺序：

date
hs300
sp500
cgb10y
gold
policy_bond
credit_bond

其中资产列为对应收益率，date 为交易日

第一行（每列）由于没有 P_{t-1}，收益率必为 NaN，保留即可

------------------------------------------
4）质量检查（必须打印）
------------------------------------------
打印以下内容：

A. 数据起始/结束日期
B. 各列缺失值统计（NaN count）
C. 各列“首次非空收益率日期”（first valid index）
D. 是否存在 inf/-inf（如果存在，将其替换为 NaN，并打印数量）
E. 是否存在价格<=0导致的 NaN（打印每列数量）

------------------------------------------
5）保存
------------------------------------------
保存为：
processed_data/asset_return_panel.csv

保存时：
- date 为第一列
- 不要保存 index
- 保留 NaN
- 数值列保持 float 类型

------------------------------------------
额外要求（代码风格）
------------------------------------------
写成一个脚本文件即可（例如 build_asset_return_panel.py）
逻辑函数化：load -> compute_returns -> validate -> save
不要引入不必要的第三方库（pandas、numpy足够）
