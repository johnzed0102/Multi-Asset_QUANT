目标：

构建一个“月度调仓 + 风险平价 + 动态资产池 + 目标波动率缩放”的回测框架。

------------------------------------------------------------
一、数据读取
------------------------------------------------------------

读取：

processed_data/asset_return_panel.csv

要求：

- date 转为 datetime
- 升序排序
- 回测全区间使用 2016-01-04 起数据
- 绩效统计从 2018-01-01 开始

资产列：

hs300
sp500
cgb10y
gold
policy_bond
credit_bond

------------------------------------------------------------
二、调仓规则
------------------------------------------------------------

- 调仓频率：每月最后一个交易日
- 识别方法：按 date 分组 year-month，选每月最后一天
- 仅在调仓日更新权重
- 其他日期沿用上期权重

------------------------------------------------------------
三、滚动窗口参数
------------------------------------------------------------

- 协方差估计窗口：252 个交易日
- 若不足 252 日，不进行优化（权重设为 NaN）
- 从 2016 开始滚动，为 2018 提供成熟窗口

------------------------------------------------------------
四、动态资产池规则（非常重要）
------------------------------------------------------------

在每个调仓日：

仅使用“当日及过去252日内有非NaN收益”的资产

例如：

- 2016–2022.06 期间，不包含 credit_bond
- 2022.07 之后，credit_bond 自动纳入

不得人为填补 NaN。

------------------------------------------------------------
五、风险平价权重计算
------------------------------------------------------------

使用“等风险贡献”原则：

步骤：

1. 计算过去 252 日协方差矩阵 Σ
2. 初始权重设为等权
3. 使用数值优化（如 scipy.optimize.minimize）求解：

目标：各资产风险贡献相等

风险贡献定义：

RC_i = w_i * (Σw)_i / (w^T Σ w)

约束：

- 权重 >= 0
- 权重之和 = 1

------------------------------------------------------------
六、目标波动率缩放
------------------------------------------------------------

目标年化波动率：

target_vol = 0.05

计算：

realized_vol = sqrt(252 * w^T Σ w)

缩放系数：

scale = min(1, target_vol / realized_vol)

最终权重：

w_scaled = w * scale

剩余权重视为现金：

cash_weight = 1 - sum(w_scaled)

------------------------------------------------------------
七、权重平滑机制
------------------------------------------------------------

为避免跳变，加入缓冲：

w_t = 0.7 * w_{t-1} + 0.3 * w_new

首个有效调仓日不做平滑。

------------------------------------------------------------
八、每日组合收益计算
------------------------------------------------------------

每日：

portfolio_return_t = sum(w_{t-1} * asset_return_t)

注意：

使用“昨日权重”计算今日收益。

------------------------------------------------------------
九、输出结果
------------------------------------------------------------

生成以下文件：

- 每个调仓日权重
- 包含 cash 列

回测引擎/backtest_results.csv

- 验证区间：2024-01-01 至 2025-12-01
- 输出：
    年化收益
    年化波动率
    夏普比率
    最大回撤
    Calmar
